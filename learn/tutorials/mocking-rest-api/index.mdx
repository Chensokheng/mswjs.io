## Introduction

In this tutorial we are going to build a TODO list application.

## Goal

By the end of this tutorial you will be able to:

1. Setup Mock Service Worker in your project for in-browser development and integration tests.
1. Write request handlers to capture any outgoing requests.
1. Mock basic CRUD operations.
1. Override mocked responses on a per-test basis.
