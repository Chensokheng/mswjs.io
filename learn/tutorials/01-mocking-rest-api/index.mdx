## Goal

By the end of this tutorial you will be able to:

1. Setup Mock Service Worker in your project for in-browser development and integration testing.
1. Write request handlers to capture any outgoing REST API requests.
1. Mock responses to CRUD operations in your application.
1. Control mocked responses on a per-test basis to test against the loading and error states of the API.
